## рџ§  How It Works

The generator uses a **syllable-based method** вЂ” for each language, there are:

- **syllable sets** (`ar`, `el`, `an`...)
- **endings** (`a`, `e`, `ing`...)
- **special symbols** (can be toggled on/off)

The algorithm combines syllables, adjusts to the selected length, adds symbols (if enabled), and capitalizes the first letter. The result is **pronounceable, readable nicknames** that sound like real names or logins.
